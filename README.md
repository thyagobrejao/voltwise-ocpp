# VoltWise OCPP

OCPP 1.6 and 2.0 server implementation for VoltWise.

This service is responsible for communication with EV chargers using the Open Charge Point Protocol.

## 🔌 Features (planned)

- BootNotification
- StatusNotification
- StartTransaction
- StopTransaction
- MeterValues
- Heartbeat

## 🛠️ Tech Stack

- Go (Golang)

## 🧩 Role in Architecture

Acts as the communication layer between physical chargers and the VoltWise cloud.

## 🚀 Goal

Provide a reliable and scalable OCPP server for EV charging infrastructure.
